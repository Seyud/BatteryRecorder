# 校准说明

## 入口

进入设置页，在“常规”分组里，你会看到这些选项：

- 串联双电芯
- 放电也显示正值
- 电流单位校准
- 电池状态校准

## 什么时候需要校准

如果你遇到这些情况，优先考虑校准：

- **首次使用软件**
- 功率数值明显异常
- 功率数值为**0.00W**
- 当前记录状态总是显示 **未知** / **未充电**
- 总是无记录文件

## 校准会影响什么

电流校准影响的是 App 内展示的功率换算结果。  
如果显示口径不对，软件所有功能都将无法正常使用。

状态校准影响的是记录器对电池状态的判定。  
如果判定错误，可能导致总是无记录文件。

## 配置

### 串联双电芯

用于匹配双电芯设备的显示换算。  
如果你的设备不是这一类，不要随意打开。

### 放电也显示正值

这个选项只改变展示时的正负号表现，方便阅读。  
它不会改变记录本身的采样逻辑。

### 电流单位校准

这个选项会改变电流的换算口径，影响功率数值的大小。  
如果你发现功率数值明显异常，或者为 0.00W，可以尝试调整这个选项来修正显示。

### 电池状态校准

这个选项会影响记录器对电池状态的判定。  
如果你发现总是无记录文件或当前记录状态总是显示 **未知** / **未充电**，可以尝试在异常持续时通过这个选项来修正判定。
